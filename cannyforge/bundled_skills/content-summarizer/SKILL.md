# Content Summarizer

## Capabilities
- Text summarization
- Key point extraction
- Length adaptation
- Citation formatting
- Quality assessment

## Usage
Provide content to be summarized along with any length or format preferences.
The skill extracts key points and produces concise summaries.

## Examples
- "Summarize this research paper"
- "Extract the key points from the meeting notes"
- "Condense this article to 3 bullet points"
